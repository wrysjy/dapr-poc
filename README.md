# Dapr with Docker-Compose

## Prerequisites
- [Docker](https://docs.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/install/)

## Running Dapr

you can just use Docker Compose directly:
```
docker-compose up
```

## Clean up

To tear down the Docker Compose deployment, you can run:
```
docker-compose down
```
